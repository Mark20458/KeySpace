### 密码管理器

- [ ] 密钥管理（增删改查）
- [ ] 进入多任务界面，页面变为空白；防止截图
- [ ] 密码生成
- [ ] 文件夹模式
- [ ] 云端同步（登录、注册、更新）
- [ ] 自动填充服务
- [ ] 搜索功能

注：需要使用[CloudStore](https://github.com/Mark20458/CloudStore)
可以通过ngrok工具，将CloudStore项目代理到公网上，然后设置KeySpace下Api类BASE_URL变量为自己的基地址
